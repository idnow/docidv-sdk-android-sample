# DocIDV Sample App

A sample Android application demonstrating the basic integration of the IDnow DocIDV SDK.

> This sample does not cover NFC feature, except german eID.

## Getting Started

Clone the repository and open it in your IDE:
```bash
git clone https://github.com/idnow/docidv-sdk-android-sample.git
```

Then run the app on a device or emulator.

> **Note:** x86-based devices and emulators are not supported.

---

## What's Demonstrated

- SDK initialization with **IDnowDocIDVConfig**
- Starting an identification session with **IDnowDocIDV**
- Handling session results via **ResultListener** (**FINISHED**, **CANCELLED**, **ERROR**)

---

## Dependencies

This sample uses the IDnow DocIDV SDK. See the [official documentation](https://www.idnow.io/developers/) for integration details.
